# Backtesting Crypto Trading Strategies with Python

A Google Colab notebook for backtesting crypto trading strategies using historical price data from the CoinGecko API — no local setup required.

## What's inside

| Notebook | Purpose |
|----------|---------|
| `guides/CoinGecko_API_Key_Google_Colab.ipynb` | Setting up your CoinGecko API key securely in Google Colab Secrets |

## What it covers

- Configuring CoinGecko API access in Google Colab without exposing your key
- Fetching historical OHLC price data for backtesting
- Implementing and evaluating trading strategies (buy the dip, golden cross, and more)
- Measuring strategy performance with standard metrics (returns, drawdown, win rate)

## Requirements

- A Google account (for Google Colab — free, no local install needed)
- CoinGecko API key ([get one here](https://www.coingecko.com/en/api))

## Getting started

1. Upload the `.ipynb` file to Google Drive
2. Open with Google Colaboratory
3. Add your CoinGecko API key to Colab Secrets (key name: `CG_API_KEY`)
4. Run all cells
