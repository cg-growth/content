## AI Cryptocurrency Assistant

This is a lightweight SvelteKit + Ts project that uses OpenAI FunctionCalling to fetch and return the latest
cryptocurrency price performance using the CoinGecko API.

Example:
![start example.png](static%2Fstart%20example.png)

It can be easily extended by adding additional functions.

### Get started

`npm i `
`npm run dev`

### Considerations

The AI will occasionally hallucinate so do bear that in mind. Additional features could include:

* Searching the web for relevant information on specific projeccts
* Placing trades on your behalf
* PRs open
