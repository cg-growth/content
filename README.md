# CoinGecko API — Content

Companion code for [CoinGecko's API guides and Learn articles](https://www.coingecko.com/learn/category/api). Each folder maps to a published article and contains working, runnable examples.

## Structure

Folders are named after the article's URL slug:

```
content/
├── x402-pay-per-use-crypto-api/
├── crypto-top-gainers-losers-python-bot/
└── ...
```

Each folder is named after its article's URL slug. Browse all API guides at [coingecko.com/learn/category/api](https://www.coingecko.com/learn/category/api).

## Getting started

Clone the repo, navigate to the relevant folder, and follow the README inside.

```bash
git clone https://github.com/cg-growth/content.git
cd content/<article-slug>
```
